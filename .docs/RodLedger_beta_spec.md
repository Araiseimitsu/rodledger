# RodLedger β - 材料管理デモ版仕様書

## 概要

NC旋盤で使用する棒材の発注・入庫・持ち出し・戻しを履歴ベースで管理し、
在庫数量および金額を可視化するデモアプリ。

------------------------------------------------------------------------

## 対象範囲（Demo）

-   材料：SUM22
-   直径：φ8.0
-   長さ：2500mm
-   単一材料のみ管理

------------------------------------------------------------------------

## コンセプト

在庫ではなく「履歴」を管理する。 在庫は履歴の結果として算出する。

------------------------------------------------------------------------

## 管理単位

-   材料（material）
-   ロット（lot）
-   トランザクション（transaction）

------------------------------------------------------------------------

## データ構造

### materials

-   id
-   name
-   diameter
-   length
-   density
-   weight_per_unit

------------------------------------------------------------------------

### lots

-   id
-   material_id
-   lot_code
-   unit_price（円/kg）
-   created_at

------------------------------------------------------------------------

### transactions

-   id
-   material_id
-   lot_id
-   type（in / out / return / adjust）
-   quantity（本数）
-   weight（kg）
-   unit_price（履歴保持用）
-   created_at
-   memo

------------------------------------------------------------------------

## 在庫計算

在庫重量 = Σ(入庫 + 戻し) - Σ(出庫)

在庫金額 = Σ(入庫金額 + 戻し) - Σ(出庫金額)

------------------------------------------------------------------------

## 重量換算

重量(kg) = 長さ(mm) × weight_per_mm

※ weight_per_mmは材料ごとに定義

------------------------------------------------------------------------

## イベント種別

-   入庫（in）
-   出庫（out）
-   戻し（return）
-   修正（adjust）

------------------------------------------------------------------------

## UI要件

### ダッシュボード

-   現在在庫（kg / 本数）
-   在庫金額

### 入力画面

-   イベント種別選択
-   lot選択
-   数量 or 重量入力
-   自動換算表示

### 履歴画面

-   時系列表示
-   編集可能
