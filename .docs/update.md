## 2026-03-31

- 出庫時の本数上限を帳簿上の本数ではなく、ロットごとの実効本数（帳簿本数と重量÷単重の換算の大きい方）で判定するようにした。残本数 0・残重量が単重以上ある場合でも、重量に見合う範囲で出庫できる。
- 在庫 API に `total_effective_quantity`（実効本数合計）と `weight_per_unit` を追加し、ダッシュボードの「本」表示は実効合計を優先するようにした。

## 2026-03-26

- backend の CORS 設定を環境変数で拡張できるようにし、README に LAN 公開時の `--host 0.0.0.0` と `CORS_ALLOW_ORIGINS` 設定例を追記した。
